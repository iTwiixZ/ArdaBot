Politique de confidentialité – ArdaBot

Dernière mise à jour : avril 2025

1. Données collectées

ArdaBot collecte et stocke les données suivantes :

  • Identifiants utilisateurs Discord
  • Pseudos en jeu, Twitch, et YouTube (si fournis)
  • ID des serveurs et de certains salons configurés

2. Utilisation des données

Les données collectées sont utilisées uniquement pour fournir les fonctionnalités du bot (XP, configuration serveur, interactions personnalisées, etc.). Aucune donnée n’est exploitée à des fins commerciales.

3. Partage des données

Aucune donnée personnelle n’est vendue ni partagée avec des tiers. Les données peuvent être transmises uniquement si la loi l’exige.

4. Suppression des données

Vous pouvez demander la suppression de vos données personnelles en rejoignant le serveur de support : https://discord.gg/DztgSf3csj

5. Sécurité

Toutes les données sont stockées de manière sécurisée dans une base MySQL et ne sont accessibles que par l’équipe de développement.

Pour toute question, merci de rejoindre le serveur de support : https://discord.gg/DztgSf3csj
